
a)
JavaScript Object Notation

b)
JSON hat die Aufgabe, strukturierte Daten in einem leicht lesbaren 
Format zu speichern und zu transportieren. 
Es wird häufig in der Kommunikation zwischen Servern und 
Webanwendungen verwendet.

c)
- Leichte Datenübertragung, weil für jede Programmiersprache
eine entsprechende Schnittstelle existiert.
- Geringe Dateigröße im vergleich zu XML
- Besser Lesbar im Vergleich zu CSV

d)
